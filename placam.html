Z790 Gaming X AX Socket 1700

Socket LGA 1700

Esta placa madre está diseñada específicamente para soportar procesadores de la serie Intel de 12ª generación, que utilizan el nuevo socket LGA 1700. Esto asegura compatibilidad con las últimas CPU de Intel y posiblemente con futuras actualizaciones.

Chipset Z790

El chipset Z790 proporciona una serie de características y funcionalidades avanzadas, incluyendo un mayor número de puertos USB, soporte para overclocking, múltiples ranuras PCIe para tarjetas gráficas y otros dispositivos, así como tecnologías de almacenamiento rápido como PCIe 4.0.

Factor de Forma ATX

Esta placa madre sigue el factor de forma ATX estándar, lo que significa que debería encajar en la mayoría de los casos de torre media o grande. El factor de forma ATX también ofrece una buena cantidad de espacio para componentes adicionales y ventilación.

Capacidad de Memoria RAM

La Gigabyte Z790 Gaming X AX probablemente admita una gran cantidad de memoria RAM DDR5, que es el estándar para la serie de procesadores de 12ª generación de Intel. Es probable que tenga múltiples ranuras DIMM para permitir una expansión significativa de la memoria.

Conectividad y Características

Esta placa madre probablemente esté equipada con una variedad de puertos de E/S, incluyendo puertos USB 3.2 Gen 2, puertos USB-C, conectores de audio de alta calidad, puertos Ethernet, Wi-Fi integrado, y posiblemente Bluetooth. También puede incluir características adicionales orientadas al gaming, como iluminación RGB personalizable, disipadores de calor robustos y soporte para tecnologías de audio avanzadas.
